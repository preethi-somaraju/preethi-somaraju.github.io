# Sales Data Reporting Pipeline & SQL Analytics

This repository houses an automated data collection pipeline that aggregates transaction records and runs analytical queries for business dashboard visualization.

## 🛠️ Pipeline Details
*   **ETL Script**: `data_pipeline.py` generates clean structured records and outputs to an SQLite database.
*   **Reporting Queries**: `sales_queries.sql` calculates growth metrics, top categories, and VIP customer tiers.
