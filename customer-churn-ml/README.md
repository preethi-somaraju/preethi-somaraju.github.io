# Customer Churn Prediction Machine Learning Model

This project builds a predictive classifier model using a **Random Forest Classifier** in Scikit-Learn to detect subscribers likely to churn.

## 📊 Key Highlights
- **Pre-processing**: Categorical encoding, numeric conversion, standard scaling.
- **Model**: Random Forest Classifier with grid-search cross-validation.
- **Metrics**: Optimized for **Recall** to capture maximum churn risks, achieving an ROC-AUC of 0.84.

## 🏃 Running the Code
```bash
python churn_prediction.py
```
